Maven Use Maven to get WebMotion

WebMotion is available with maven by adding the following lines to your projects's pom.xml file:

<dependency>
    <groupId>org.debux.webmotion</groupId>
    <artifactId>webmotion</artifactId>
    <version>2.5</version>
</dependency>

Netbeans Plugin Simplify the use of WebMotion

The plugin is available on the Netbeans website. It offers the following functionalities:

  • coloriation
  • syntax checking
  • folding
  • completion help
  • link to the files
  • hints
  • refactoring
  • templates
  • comment/uncomment

The source code of the plugin is available under Github.

Sources Get the sources

The sources are available under Github, you can anonymously checkout the project by executing the following comand:

$ git clone git@github.com:webmotion-framework/webmotion.git webmotion.git

or

$ git clone https://github.com/webmotion-framework/webmotion.git webmotion.git
