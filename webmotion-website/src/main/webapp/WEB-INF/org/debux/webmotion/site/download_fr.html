Maven Utiliser maven pour récupérer WebMotion

WebMotion est disponible par maven en ajoutant les lignes suivantes dans votre pom.xml de votre projet :

<dependency>
    <groupId>org.debux.webmotion</groupId>
    <artifactId>webmotion</artifactId>
    <version>2.5</version>
</dependency>

Plugin Netbeans Simplifier l'utilisation de WebMotion

Le plugin est disponible sur le site de Netbeans. Il comporte les fonctionnalités suivantes :

  • colorisation
  • vérification syntaxique
  • folding
  • aide à la complétion
  • lien vers les fichiers
  • hints
  • refactoring
  • templates
  • comment/uncomment

Le code source du plugin est disponible sur Github.

Aperçu du plugin

Sources Récupérer les sources

Les sources sont disponibles sur Github, vous pouvez les récupérer en accès anonyme en exécutant la commande suivante :

$ git clone git@github.com:webmotion-framework/webmotion.git webmotion.git

ou

$ git clone https://github.com/webmotion-framework/webmotion.git webmotion.git
